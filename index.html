Organize your club's floorball events.

HelperZone is an application which helps Swiss floorball clubs to organize their SwissUnihockey league games and tournaments.

Go to Github


Connected to Swissunihockey API.

HelperZone uses data through the official Swissunihockey API. There is no need for the administrator to input game events by himself.

Go to Github

Event Planning

Organize your club's floorball tournaments. Decide which helper tasks are needed. Teams and team members can then be assigned to these tasks.

User Management

Manage your club's members and allocate them to the teams. Allow your members to access the application with their own login.

Reports

Use the report functionality to print posters for dressing rooms (incl. allocation algorithm), drink/food vouchers, match schedule.

Import your club's events. Only one click away.
